# Knowledge Discovery & Pattern Extraction

Conversational Graph Extraction From Unstructured Data

## Setup

### Requirements

- `.env` with your keys
  - `MISTRAL_API_KEY=...`
  - `HF_TOKEN=...`
- `aria2`
- `uv` python package manager
- `python>=3.14`


## Usage

```shell
uv sync
```

```shell
uv run data/download.py
```

```shell
uv run src/main.py
```
